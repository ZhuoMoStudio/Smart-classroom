# Smart-classroom.
灵动课堂 (Smart Classroom)

https://img.shields.io/badge/platform-Windows%20%7C%20Linux%20%7C%20macOS%20%7C%20Android-lightgrey
https://img.shields.io/badge/python-3.8%2B-blue
https://img.shields.io/badge/license-MIT-green
https://img.shields.io/badge/Kivy-2.3.0-66cc00

一个极简、便携、跨平台的课堂互动工具，专为教师设计。
支持随机抽学生/小组、倒计时器、积分排行榜、题库抽题（含风险题过滤），以及WebDAV 云同步。
数据存储在程序同级的 数据 文件夹中，拷贝到 U 盘即可随身携带，即插即用。

---

✨ 功能特性

· 🎲 随机抽取：支持抽取单个学生或整组，带动画滚动效果。
· ⏱️ 倒计时器：预设 3/5/10 分钟快速设置，时间到自动变色提示。
· 📊 积分排行榜：点击分数可直接编辑，一键 +1 加减分，根据积分自动显示段位徽章。
· 📚 课堂题库：支持 CSV 批量导入题库，随机抽题。自动识别风险题（第三列），风险题显示红色边框，默认不参与随机抽取，安全可控。
· 💾 U 盘便携：所有数据（班级、积分、题库、配置）保存在程序同级的 数据 文件夹中，直接拷贝文件夹即可完整迁移。
· ☁️ WebDAV 云同步：支持坚果云等 WebDAV 服务，手动上传/下载数据库，多设备间同步积分和题库。
· 🌐 中英双语：一键切换中文 / English，界面即时刷新。
· 🧑‍🏫 新手引导：首次启动展示操作提示，帮助教师快速上手。
· 🏫 模拟班级：内置“生成模拟班级”功能，一键生成测试数据，方便体验。
· 📂 一键导入：在设置中选择外部文件夹，自动合并 JSON 数据文件并导入所有 CSV 题库。


🚀 快速开始

环境要求

· Python 3.8 或更高版本
· Kivy 2.3.0 （推荐使用虚拟环境）

安装依赖

```bash
pip install kivy==2.3.0
```

运行程序

```bash
python smart_classroom.py
```

首次启动会自动生成默认班级和示例题库，并弹出新手引导。

打包为独立程序（可选）

可以使用 PyInstaller 将脚本打包成 exe（Windows）或可执行文件（Linux/macOS），打包后仍会保持 数据 文件夹的相对路径，实现便携性。

📝 数据说明

数据存储位置

所有用户数据（班级、积分、题库、设置）保存在程序同级的 数据/smart_classroom_v2.json 中。
拷贝整个 数据 文件夹即可完整迁移到其他电脑或 U 盘。

CSV 题库格式

题库文件必须为 .csv 格式，UTF-8 编码，包含表头行（可省略，但建议保留）。
列结构如下：

列序号 含义 必填 说明
1 题目正文 是 题目内容
2 答案 否 答案文本，抽取后显示
3 风险标记 否 标记为“是”、“true”、“1”视为风险题；其他值或留空视为普通题。缺省为普通题。

示例 CSV 文件：

```csv
题目,答案,是否风险
中国的首都是哪个城市？,北京,否
光合作用的主要产物是什么？,葡萄糖和氧气,否
《静夜思》的作者是谁？,李白,是
地球绕太阳公转一周大约需要多少天？,365天,否
水的化学式是什么？,H₂O,否
三角形内角和等于多少度？,180°,是
Python 中用于定义函数的关键字是什么？,def,否
```

风险题特殊处理

· 在题库抽题时，默认排除所有风险题，确保课堂安全。
· 若手动抽到风险题（未来版本可切换），题目区域会显示 红色边框 和 “⚠ 风险题” 警示文字。
· 导入时自动识别，无需额外配置。

🔧 高级功能

文件夹批量导入

1. 点击右上角 ⚙ 设置 → 从外部文件夹导入数据和题库。
2. 选择一个包含 .json 数据文件和/或 .csv 题库文件的文件夹。
3. 程序会自动：
   · 用找到的 JSON 文件覆盖当前班级和积分数据。
   · 将所有 CSV 文件合并到题库列表中（文件名作为题库名称）。
4. 导入成功后会弹出提示并刷新界面。

WebDAV 云同步

支持坚果云等 WebDAV 网盘，实现多设备间数据同步。

1. 在设置中填写服务器地址、用户名、应用密码（坚果云需在账户安全中生成第三方应用密码）。
2. 点击 上传到云端 将本地数据推送至云端的 /smart_classroom_sync/ 目录。
3. 其他设备点击 从云端下载 拉取最新数据，自动加载。
