# 更新日志

## v2.0.0 (2026-07-04)

### ✨ 重大更新
- **版本升级 1.0.8 → 2.0.0**：全面功能完善

### 🎨 标注系统重大升级
- **图层管理**：多图层叠加（添加/删除/可见/锁定/重命名/排序），每页独立图层
- **蒙层功能**：6种揭示模式（左到右/右到左/上到下/下到上/中心扩散/点击揭示），进度可调
- **荧光笔**：半透明宽笔触，5种荧光色
- **撤销/重做**：完整 undo/redo 支持（上限50步）
- **标注持久化**：支持 alongside（与PDF同目录）和 appPrivate（应用私有）两种存储模式，自动保存/加载
- **保存按钮**：标注模式顶部提供手动保存入口

### 📚 教材仓库重构
- **内部组件化**：移除独立浏览器页面，改为内嵌面板（首页第5个Tab）
- **本地索引**：目录结构缓存本地，无需网络即可浏览
- **按需下载**：选择教材后才下载 PDF，节约流量
- **下载管理**：已下载标记 + 下载统计（已下载/总数）
- **刷新机制**：点击右上角刷新获取最新目录

### 📱 自适应布局完善
- **iPad SplitView 风格**：平板端左侧导航栏（56px图标列）+ 右侧内容区
- **iOS 风格分组列表**：设置/更多页面使用 iOS Settings 风格分组卡片
- **教材 Tab 原生集成**：手机端6Tab、平板端侧栏均可直达
- **响应式优化**：所有面板适配平板/手机双模式

### 🎯 iOS 主题深化
- **透明导航栏**：AppBar 透明背景 + scrolledUnderElevation 0.5
- **iOS 分组列表**：带标题的分组卡片（更多页面）
- **底部 Sheet**：大圆角模态 bottom sheet
- **Dialog 大圆角**：统一 20px 圆角
- **SnackBar 浮动**：floating 行为 + 大圆角
- **日期选择器/进度条**：主题统一
- **NavigationBar 透明**：底部栏半透明毛玻璃效果

### 🐛 修复/改进
- 多处 import 修复
- 标注手势与蒙层手势协调
- PDF 阅读器退出时自动保存标注

### 🔧 数据管理重构
- **新增 DataService**：集中化保存/加载逻辑，防抖自动保存（dirty后5秒内无修改才写入）
- **生命周期保存**：应用进入后台时自动立即保存
- **Provider 驱动**：`autoSaveProvider` 监听 dirty 状态自动触发防抖保存
- **移除旧 autoSaveTimer**：不再使用固定间隔轮询，改为事件驱动

### 🔘 按钮合并优化
- **导入/导出合并**：5个独立按钮 → 1个「导入/导出数据」统一入口，弹出 iOS 风格底部菜单
- **CentralConsole 精简**：10个回调 → 6个核心入口，导入导出通过底部菜单整合
- **节省空间**：手机端更多面板从 12 个条目减至 7 个，布局更清晰
- **桌面端同步**：平板更多面板同样合并，双栏更均衡
